# WSB Stock Classifier

**A GPT-powered sentiment engine that analyzes WallStreetBets posts to gauge market vibes.**

🚀 **[Live Demo](https://stock-classifier--tarunmanjunath1.replit.app)**

---

### What it Does

WSB Stock Classifier aggregates historical Reddit data to classify investor sentiment as **Positive**, **Negative**, or **Neutral**. By mapping retail discussion volume and post flairs, it helps investors spot trending tickers and shifting market momentum at a glance.

### How it Works

The app parses a curated dataset of forum text and feeds it into an advanced LLM/GPT sentiment engine. The model evaluates the underlying tone, assigns a sentiment category, and outputs real-time dashboard analytics, highlighting "Hot Tickers" alongside a crowd-sourced "Sentiment Pulse."

### Stack

* **Frontend/Backend:** Python, Streamlit
* **AI Engine:** OpenAI GPT API / LLM
* **Hosting:** Replit
